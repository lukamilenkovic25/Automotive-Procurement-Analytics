# Automotive-Procurement-Analytics
Python portfolio project with data cleaning, visualization, KPI dashboard and reporting. This project analyzes automotive procurement data from 2020 - 2025 using Python (Pandas and Matplotlib). The objective is to transform raw procurement data into business insights through data cleaning, exploratory analysis, KPI reporting and data visualization.


## Project Overview
This project analyzes automotive procurement data from 2020 to 2025 using Python (Pandas and Matplotlib).
The objective is to transform raw procurement data into business insights through data cleaning, exploratory analysis, KPI reporting and data visualization.
The dataset contains project information related to:
  - OEM customers
  - Suppliers
  - Commodity groups
  - Plant locations
  - Annual volumes
  - Target prices
  - RFQ, Nomination and SOP milestones

## Objective
The main goal of this portfolio is to:

- Create procurement KPI dashboards

## Analysis Performed
### Data Quality Checks
- Dataset dimensions
- Missing values analysis
- Duplicate records analysis
- Date validation
### Time-Based Analysis
- RDQ projects per year
- RFQ projects per month
- Procurement activity trends (2025-2025)
### Supplier Analysis
- Projects per supplier
- Average annual volume
- Average target price
- Top suppliers by project count
### OEM Analysis
- Project distribution by OEM
### Commodity Analysis
- Project distribution by commodity group
### Procurement KPI Dashboard
- Number of projects
- Average volume
- Average target price
